Some Logos of TeX

今天看见 GitHub 上一个 LaTeX-Example，我突然也想要写一些 LaTeX Example，于是就写一些 LaTeX 的 Logo 吧，也就是那些错落有致的符号。

第一次看见它们可能有点奇怪，不过真是越看越好看，也许这就是 LaTeX 的魅力所在吧，这些符号也是 LaTeX 区别于其他排版系统或软件的最大不同之处之一了，Happy LaTeXing!

可前往 查看更多或者下载它们。代码很简单，如下所示：

\documentclass{standalone}

\usepackage{newtxtext}
\usepackage{hologo}

\begin{document}

    \TeX

    \LaTeX

    \LaTeXe

    \hologo{BibTeX}

    \hologo{pdfTeX}

    \hologo{pdfLaTeX}

    \hologo{XeLaTeX}

    \hologo{XeTeX}
		
    \hologo{AmSLaTeX}
		
    \hologo{NTS}
		
    \hologo{METAFONT}

\end{document}
